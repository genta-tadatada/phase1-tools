# Frontend Design Skill

Before coding, establish:
- **Purpose**: Problem solved & user context
- **Tone**: Bold aesthetic direction (minimalist, maximalist, retro, organic, luxury, playful, editorial, brutalist, etc.)
- **Constraints**: Technical requirements
- **Differentiation**: Memorable distinguishing feature

## Design Focus Areas

### 1. Typography
- Distinctive, characterful fonts (avoid Arial, Inter)
- Pair display font with refined body font

### 2. Color & Theme
- Cohesive aesthetic with CSS variables
- Dominant colors with sharp accents

### 3. Motion
- CSS-only animations for HTML
- Motion library for React
- Page load reveals with staggered delays

### 4. Spatial Composition
- Asymmetry, overlap, diagonal flow
- Grid-breaking elements
- Generous or controlled density

### 5. Visual Details
- Gradients, noise textures, patterns
- Dramatic shadows, decorative borders
- Custom cursors, grain overlays

## What NOT to Do
❌ Generic AI aesthetics
❌ Overused fonts (Inter, Roboto, Arial)
❌ Clichéd color schemes (purple gradients)
❌ Predictable layouts
❌ Cookie-cutter patterns

## Core Principle
**Match implementation complexity to vision**: Maximalist designs need elaborate animations; minimalist designs need precision and restraint. Execute the vision completely.
